# README
## Getting Started
```bash
cd src/mcp-uuid
pnpm i
pnpm build
```
Then open this directory with Cursor.
```
Cursor .
```

## Inspector
<https://github.com/modelcontextprotocol/inspector>
```
pnpm dlx @modelcontextprotocol/inspector node src/mcp-uuid/dist/index.js
```
